# AsyncStandup Bot

Telegram bot for async daily standups: DMs three configurable questions, sends one nudge, compiles tidy digests to a team channel, and keeps searchable history.

Spec: [`docs/blueprint.md`](docs/blueprint.md).

Built on [agnt-gm.ai](https://agnt-gm.ai). The whole bot is built and refined here as pull requests across successive build passes.
